# FPS Demo
![Interractions](readme-images/interactions.png)

## Controls 
- **Escape** to leave
- **WASD** to move
- **Lmouse** to shoot
- **E** to Interact

## Features
- **Shoot**: The arm can shoot, leaving "bullet holes" (**Might be loud, careful**)
- **Cat**: it spins
- **Lamp**: It glows, toggle 
- **Shower**: toggle to water
